# superturtle.dev

Landing page and docs for [Super Turtle](https://github.com/Rigos0/superturtle) — an autonomous coding system you control from Telegram.

## Structure

- `landing/` — Next.js site deployed on Vercel at [superturtle.dev](https://www.superturtle.dev)
- `docs/` — Mintlify docs served at [superturtle.dev/docs](https://www.superturtle.dev/docs)
